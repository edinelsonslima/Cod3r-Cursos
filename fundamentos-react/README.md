## Dependências 
 - react: 17.0.2
 - react-dom: 17.0.2

## Inicie a aplicação
Primeiro, rode o comundo para instalar as dependências:

```bash
npm install
# or
yarn i
```
depois inicie normalmente com **npm run start** ou **yarn start**

Abra [http://localhost:3000](http://localhost:3000) com seu navegador parar ver a aplicação.
